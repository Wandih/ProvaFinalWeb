"O que é um Servidor Web (Web Server)?"

Um Servidor Web é um dispositivo com a finalidade de armazenar, processar e entregar arquivos de sites para navegadores

Há dois tipos de servidores web:

  • Estáticos: o servidor entrega o arquivo pedido pelo cliente, sem demais ações
  • Dinâmicos: o servidor entrega dados que são atualizados constantemente a cada requisição, dados esses que são criados por ações do usuário ou por alguma aplicação.
