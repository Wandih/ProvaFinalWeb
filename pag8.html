Comandos básicos do Linux

  • cd "nomepasta/": vai até a pasta selecionada
  • ls: mostra todo o conteúdo (pastas, arquivos, etc) do diretório escolhido
  • mkdir: adiciona um diretório novo
  • rm: delete o item selecionado
  • mv: move ou renomeia um arquivo
  • sudo + * insira comando*: executa uma ação como superusuário
  • touch: pode criar um arquivo novo
