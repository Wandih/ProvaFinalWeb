"O que é um servidor de banco de dados?"

Servidor ou servidores de banco de dados se trata de aplicações ou equipamentos em um computacional utilizados para o armazenamento e gerenciamento de dados de uma maneira estruturada.

Essas aplicações podem ser:

  • Estáticas, isto é, aplicações simples que visam apresentar conteúdos concisos e pode operar de modo offline e não necessita da instalação de demais aplicações
  • Dinâmicas, isto é, aplicações que se utilizam de base de dados para que sejam carregados e estão em constante atualiações sempre que o usuários os acessa
