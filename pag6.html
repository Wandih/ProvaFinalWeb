"O que é um serviço SSH?"

SSH (Secure Shell) se trata de um protocolo de segurança para a troca de arquivos de cliente para servidor, utilizando criptografia

Com o auxílio de ferramentas como o SFTP (SSH File Transfer Protocol) e SCP (Secure Copy), ele possibilita a transferencia de arquivos de uma maquina local e o servidor
